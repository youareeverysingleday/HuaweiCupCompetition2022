#### 大致思路

解题思路

1. 题目：“病毒感染人数数据及其它附件数据或你们能搜集到的数据对长春市实行发放蔬菜包前后效果进行判别与分析”
   1. 先将新冠人数的变化趋势列出来，每个数据作为标签数据；然后将蔬菜包的各种特征数据作为输入变量。建立模型。分析数据之间的相关性。间接的相关性特征也可以。
   2. 通过复杂数据中提取投放点和投放的其他特征。
   3. 决策变量是每个地点的投放数量。
2. 题目：
   1. “投放点数量的合理性分析，并做适当的优化”
      1. 首先需要确定的是什么是“合理性”？按照附件2中的数据，仅仅是按照总人口数来设置投放点，显然是不合理的。肯定需要结合其他的变量因素来进行决策。直观的而言，单位面积中的人数显然就比总人数这个判断因子更合理。
      2. 这个问题归类为**优化问题**。参考<https://zhuanlan.zhihu.com/p/480312601>。
      3. 多阶段优化对小区的特征进行抽象。决策变量是每个地点的投放数目。优化条件是设置一个上限和下限。优化条件是数量最少、服务的效用最大。边界的辐射等。
      4. 单目标优化。将站点数量作为约束进行求解。
   2. “考虑未来疫情、自然灾害等特殊事件，对于政府储备物资和大规模物资分拣场所的位置与数量规模进行合理规划，并提出最优的选址数量、规模及其潜在的备用场所位置。”
      1. 这个问题定义为一个回归问题。
      2. 对疫情做出预测，以当前最坏情况来进行处理。
      3. 自然灾害在于“市州支援”、“市级直采”和“属地自保”三种来源的均衡分配。
      4. 对分拣场的优化-------
3. 题目：“**请依据附件5分析蔬菜包需求、发放规律**，并根据附件3中的各小区位置与人口信息，评价并调整4月10日至4月15日蔬菜包供应方案”
   1. 对蔬菜包的需求和发放做一个分析。也就是做供需关系分析。
      1. 什么地点做这个蔬菜包的供应
      2. 每个地点的供应量有多少。
   2. 调整方案，基于第2问对模型做一个改进。在模型中可以把需求和供给加上去。需求和供给可以通过采购来源地进行等特征进行调整。
4. 题目：
   1. “特殊时期保障居民生活物资供应的详细预案（有序网络图）”。第2问和第3问只是做一个整体的分配数量即可。
      1. 实际做的是一个网络优化问题。也就是每个组成的网络需要得到最优解。
      2. 每个地址都需要进行翻译，将地址前面几个相同的进行建图。
      3. 计算的方式是：“工作量按运输里程与小区居民人数乘积计算”
   2. “请进一步考虑用卡车运送物资，大卡车每辆可装10吨，小卡车每辆可装4吨,观察预案有无显著不同”
      1. 主要是考虑点在于运输量。可以做一个假设：全部使用大卡车或者全部使用小卡车的时候进行比较。
      2. 需要加一些条件限制，保证卡车的数目不能变化太大。
